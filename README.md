# Founder Intelligence Platform

Founder Intelligence Platform is a project for publishing verified, reusable founder intelligence.

The first focus area is Dubai and UAE business setup research. The public repository contains only reviewed, public-safe material.

## Public Boundary

This repository is for founder-facing outputs only:

- public documentation
- approved examples
- sample templates
- open-source tools
- reviewed product outputs

The private Research Operating System, raw sources, internal notes, interviews, source scoring, automation workflows, prompt libraries, and internal knowledge database are not published here.

## Principle

Research once. Publish everywhere.

## What Is Here

- `docs/` - public documentation about the platform and publishing boundary
- `examples/` - sanitized examples of founder intelligence outputs
- `templates/` - public templates for reusable research artifacts
- `tools/` - future open-source utilities

## Current Milestone

The public repository now includes the first public-safe milestone:

- a clear public/private boundary
- a public ROS overview
- reusable founder research templates
- a sanitized UAE ICP service-map example

## What Is Not Here

This repository does not include:

- raw research captures
- private source notes
- interview transcripts
- source scoring
- internal knowledge graph data
- private automation
- prompt libraries
- unreviewed legal, tax, visa, banking, or regulatory guidance

## Status

Stage 1 to Stage 2 transition: private build mode with selected public-safe examples.

Public outputs are added only after review.

## Founder-Facing Scope

The platform is designed to help founders answer practical questions about:

- company setup
- free zones
- banking
- visas and residence
- tax and compliance
- operating decisions

Public material should be treated as structured research support, not legal, tax, banking, immigration, or investment advice.
