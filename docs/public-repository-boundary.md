# Public Repository Boundary

This repository contains only public-facing material.

## Allowed

- Reviewed founder-facing documentation
- Approved examples
- Sample templates
- Open-source tools
- Public product outputs

## Excluded

- Raw research
- Private source notes
- Interview transcripts without explicit permission
- Source scoring
- Internal knowledge graph data
- Automation internals
- AI prompt libraries
- Internal operating notes
- Unverified legal, tax, visa, or banking claims

## Promotion Rule

Material may move from the private repository to the public repository only when it has been reviewed, cleaned of private material, and written in founder-safe language.
